<h1>🎀 Convite para Kuromi 🎀</h1>

Um aplicativo móvel interativo e temático desenvolvido em **React Native** e **TypeScript**, inspirado no universo da personagem Kuromi, do universo da Hello Kitty. O app foi estruturado para exibir uma interface dinâmica de convite, como se você pudesse convidar a própria Kuromi para uma festa.

---

### 🎓 Proposta do Projeto
O projeto foi desenvolvido como uma das atividades do curso "Formação React Native Developer" da DIO - Digital Innovation One. Do qual a ideia inicial era ser um aplicativo do Batman, como um BatSinal, para alertar o Batman sobre algum ato criminoso na cidade de Gotham. Contudo, a ideia foi transformada em um aplicativo de convite de festas para a Kuromi.


## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

- **React Native** – Framework para desenvolvimento mobile multiplataforma (iOS/Android).
- **TypeScript** – Superset JavaScript que adiciona tipagem estática e segurança ao código.
- **Expo (opcional/base do projeto)** – Ferramenta para facilitar a execução, testes e build do app.

---

## 📁 Estrutura de Arquivos

Abaixo está a organização das principais pastas e componentes do sistema:

```text
├── assets/             # Imagens, ícones e fontes temáticas da Kuromi
├── src/                # Código fonte principal do aplicativo
│   └── screens/        # Telas de navegação do usuário
│       └── home/       # Tela principal do convite interativo
├── App.tsx             # Componente raiz do React Native
├── app.json            # Configurações globais do aplicativo (metadados e ícones)
├── index.ts            # Ponto de entrada da aplicação
└── tsconfig.json       # Configurações do compilador TypeScript
```

## 🛠️ Como Executar o Projeto

Siga os passos abaixo para rodar a aplicação localmente em sua máquina.

### Pré-requisitos
Antes de começar, certifique-se de ter instalado:
* **Node.js** (versão LTS recomendada)
* **Gerenciador de pacotes** (`npm` ou `yarn`)
* **Emulador** (Android Studio / Xcode) ou o aplicativo **Expo Go** no celular.
